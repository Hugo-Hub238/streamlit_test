# Análisis de Vehículos en EE.UU.

## Descripción
Aplicación web interactiva construida con Streamlit para explorar 
el conjunto de datos de vehículos usados en EE.UU.

## Funcionalidad
- Visualiza las primeras filas del dataset
- Genera histogramas interactivos de precio y kilometraje con un clic
- Gráficas construidas con Plotly Express

## Cómo ejecutar
pip install -r requirements.txt
streamlit run apli_uno.py
